# 🚀 TaskFlow - React To-Do App

Uma aplicação de gerenciamento de tarefas desenvolvida para consolidar conceitos fundamentais de **React.js** e **TypeScript**, focada em manipulação de estados complexos e design responsivo.

[Clique aqui para visitar a página](https://to-do-list-pdgxwy95g-marlissonvinicius-projects.vercel.app/)
---

## 🧠 Conhecimentos Aplicados

### ⚡ React & Lógica de Frontend
* **Gerenciamento de Estado Imutável**: Uso do hook `useState` para controlar listas de objetos e formulários dinâmicos sem mutação direta de dados.
* **Manipulação de Eventos**: Implementação de `onSubmit` e `onChange` com tratamento de eventos para capturar dados do usuário e prevenir o comportamento padrão do navegador.
* **Renderização Dinâmica**: Uso da função `.map()` para transformar arrays de dados em componentes visuais, garantindo uma interface reativa.
* **Identificadores Únicos**: Utilização da API nativa `crypto.randomUUID()` para geração de IDs únicos, garantindo a integridade das chaves (`keys`) no React.

### 🎨 CSS & UI/UX
* **CSS Grid & Flexbox**: Estrutura responsiva que organiza os cards de tarefas de forma fluida em diferentes resoluções.
* **Feedback Visual**: Implementação de microinterações (hover e transições) para melhorar a usabilidade e o experiência.

---

## 🛠️ Tecnologias Utilizadas

* **Vite(v7.3.1)**: Ferramenta de build de última geração para um desenvolvimento rápido.
* **React (v19.2.7)**: Biblioteca principal para construção da interface de usuário.
* **TypeScript(v5.9.3)**: Linguagem para adicionar tipagem estática, reduzindo erros e aumentando a produtividade.
* **Bun(v1.3.10)**: Runtime "all-in-one" de alta performance utilizado para acelerar a execução do projeto.
